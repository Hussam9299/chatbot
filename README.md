# Panaversity AI Assistant

A chatbot built with Chainlit and Google's Gemini AI that can process text and images.

## Features

- Chat with AI assistant using Gemini 2.0 Flash
- Upload and process text files
- Upload and analyze images (extract text from images)
- Real-time conversation with chat history
